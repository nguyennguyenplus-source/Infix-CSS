01. Khái niệm & Đặc điểm

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ được sử dụng để tìm và định dạng các thành phần được tạo ra bởi các ngôn ngữ đánh dấu (như HTML).

Nó đóng vai trò là "người thợ may" thiết kế bộ trang phục lộng lẫy cho khung xương HTML của website.

02. Cấu trúc CSS

CSS Syntax

Cú pháp chuẩn

Một bộ chọn (Selector) đi kèm với một khối khai báo chứa các thuộc tính và giá trị được đặt trong dấu ngoặc nhọn.

1

Selector

Thành phần HTML bạn muốn định dạng.

2

Property & Value

Thuộc tính cần đổi và giá trị tương ứng.

h1 {
color: red;
border: 2px solid blue;
font-family: Tahoma;
}

03. Các cách triển khai

a) CSS trong (Internal)

Được đặt trong cặp thẻ <style> bên trong phần <head> của tài liệu HTML. Thích hợp cho các định dạng chỉ dùng duy nhất cho một trang. 

<!-- HTML file -->
<head>
  <style>
    body { background-color: linen; }
    h1 { color: maroon; }
  </style>
</head>

b) CSS ngoài (External)

Cách chuyên nghiệp nhất: viết CSS trong một file .css riêng biệt và liên kết nó với HTML thông qua thẻ <link>. Giúp tối ưu bộ nhớ đệm trình duyệt. 

<!-- HTML file -->
<head>
  <link rel="stylesheet" href="styles.css">
</head>

/* styles.css file */
.container { max-width: 1200px; }

c) CSS nội tuyến (Inline)

Sử dụng thuộc tính style trực tiếp trong thẻ mở HTML. Thường dùng cho các trường hợp đặc biệt cần ghi đè các định dạng khác ngay lập tức. 

<!-- HTML file -->
<h1 style="color:blue; text-align:center;">
  This is a heading
</h1>
